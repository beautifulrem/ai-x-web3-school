## 四、信息卫生纪律

- 同一观点至少 2 个独立来源交叉再相信
- 转推不等于背书,看链接里实质内容
- 代码相关:必须能跑 / 必须能查 commit / 必须能查 author
- 项目类:看 docs > 看博客 > 看 X thread > 看 KOL 二手解读
- 任何"必须立刻 ape"的紧迫感都先放冷宫 24h 再看
